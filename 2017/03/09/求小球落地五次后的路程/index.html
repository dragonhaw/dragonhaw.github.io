求小球落地五次后的路程

Mar 9, 2017

题目描述

假设一个球从任意高度自由落下，每次落地后反跳回原高度的一半; 再落下, 求它在第5次落地时，共经历多少米?第5次反弹多高？

输入描述

输入起始高度，int型

输出描述

分别输出第5次落地时，共经过多少米第5次反弹多高

输入例子

1

输出例子

2.875

0.0315

思路

在看到这种题的时候，我们脑海中应该呈现出的是画图，然后找到规律。

可以看出，除了第一次落地的路径为一次外，其他都是两次。

所以，我们的代码可以这么写：

import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        while(in.hasNext()) {
            double n = Double.parseDouble(in.next());
            System.out.println(getJourney(n));
            System.out.println(getTenthHigh(n));
        }
    }

    public static double getJourney(double high) {
        double count = high;
        for(int i = 1; i < 5; i++) {
            high = high / 2;
            count = count + high * 2;
        }
        return count;
    }

    public static double getTenthHigh(double high) {
        for(int i = 0; i < 5; i++) {
            high /= 2;
        }
        return high;
    }
}

当然，有一种很牛逼的写法：

import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        while(in.hasNext()) {
             double n = Double.parseDouble(in.next());
             System.out.println(n * 2.875);
             System.out.println(n * 0.03125);
         }
    }
}

这种是按比例缩放，当高度为1的时候，对应的路程为2.875，高度为0.03125。
